# vue-shopping
vue 前台電商購物-樂月蔬果行

| vue Version | bootstrap version | owl.carousel | vee-validate  |
| ----------- | -----------       |  ----------- | ------        |
| `2.x`       | `5.x`             | `2.x`        | `3.x`         |

## Project setup
```sh
yarn install
```

### Compiles and hot-reloads for development
```sh
yarn serve
```

### Compiles and minifies for production
```sh
yarn build
```

### Lints and fixes files
```sh
yarn lint
```

### deploy Github pages
```sh
yarn deploy
```
